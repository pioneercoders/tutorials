<summary>Write function to find given number is even or odd.</summary>
<p>

```python
   def check_even_odd(number):
    if number % 2 == 0:
        return "Even"
    else:
        return "Odd"

# Example usage:
num = int(input("Enter a number: "))
result = check_even_odd(num)
print(f"The number {num} is {result}.")
```
